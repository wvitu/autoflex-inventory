# Autoflex Inventory Control

Web system to manage products and raw materials stock, define bill of materials, and calculate production suggestions based on available inventory prioritizing higher value products.

## Requirements
- Web platform (API + Frontend)
- Responsive UI
- Database persistence (Postgres/MySQL/Oracle)
- Backend framework (Spring/Quarkus)
- Frontend framework (React/Redux suggested)
- All code and database naming in English

## Deliverables
- Backend API (CRUD: products, raw materials, bill of materials)
- Production suggestion endpoint (greedy by highest product price)
- Frontend UI for all operations
